Vue.js 备忘录

模板语法Template syntax

设置成纯Text

<span>{{ msg }}</span>

// Or

<span v-text='msg'></span>

设置成HTML元素

读取变量的时候，不会把它当作普通字符串。而是一个HTML元素进行解析

<span v-html='rawHTML'></span>

使用JS语法

<span> {{ msg.reverse() }} </span>

<span> {{ let msg = 'hi' }} </span>

指令Directives

v-if   展示el元素如果条件真
v-else-if
v-else
v-show 使用display隐藏|显示值
v-text
v-html
v-for
v-on or @
v-bind or :
v-model

v-once # 设置在模板中读取变量只读取一次，后续这个变量更改了，不会跟着变化。

有条件的呈现

添加/删除 节点

<div v-if='date == today'>
    ...
</div>
<div v-else-if='!done'>
    ...
</div>
<div v-else>
    ...
</div>

切换显示CSS而不是编辑DOM

<div v-show='date == today'>
    ...
</div>

处理事件

捕获和事件并调用方法

<div v-on:click='count'>Increase</div>
<!-- 简写 -->
<div @click='count'>Increase</div>

方法是一个原生DOM事件

count: function(event){
	console.log(event.target);
}

事件修饰符

.stop 阻止冒泡
.once 只执行一次
.prevent
.self

循环遍历

循环一个数组

<li v-for='item in items' :key='item'>
	{{ item  }}
</li>

循环带有下标

<li v-for='(item, index) in items' :key='item'>
	{{ item  }}
</li>

循环一个对象

<li v-for='obj in objs'>
	{{ obj  }}
</li>

绑定数据

简单绑定

<div v-bind:id='objId'>
    ...
</div>

<!--简写-->
<div :id='objId'></div>

绑定input标签

<input v-model='email' />
  • 标签修饰符
.lazy  转变为在 change 事件中同步,加上.lazy后相当于 双向数据绑定不起作用了
.trim 会自动过滤掉输入的首尾空格

使用对象绑定Class/Styles

<input :class='{ error: hasError }'/>
<input :style='{ margin: space+"px" }'/>

插槽 slots

插槽允许从父节点注入内容组件到子组件

基本使用

子组件MyButton.vue

<div>
	Hello World
    <slot></slot>
</div>

父组件

<my-button>
	这是一个按钮
</my-button>

命名插槽

自定义组件中可以有多个插槽。需要指定名字来区分了。没有默认为default

子组件

<div>
    <slot name='top'></slot>
    <slot name='bottom'></slot>
</div>

data(){
	return {
		topData: '头部区域',
		bottomData: "脚注区域"
	}	
}

父组件

<my-button>
    <template v-slot:top='topProps'>
        {{topProps.topData }}
    </template>
    <template v-slot:bottom>...
    </template>
</my-button>

作用域

插槽中，不能够使用组件中的数据，只能从父组件中读取数据。

子组件

<div>
    <slot v-bind:post='post'>
    {{ post.title }}
    </slot>
</div>

父组件

<my-button>
    <template v-slot:default='slotData'>
    {{ post.author }}
    </template>
</my-button>

默认值

可以在定义插槽的地方，先定义好默认值，这样以后不传递参数，也会有一个默认值。

如果默认值不符合，也可以自己设置。

生命周期函数

  • beforeCreate() Start of component
  • create() Reactive data exists
  • beforeMount() Before mounting DOM
  • mounted() DOM can be accessed
  • beforeUpdate() Still have old values
  • updated() Values have been changed
  • beforeDestroy() Component still complete
  • destroyed() Teardown complete

生命周期方法

  • $mount() Mount component to DOM
  • $forceUpdate() Force re-render
  • $nextTick() Runs func next update
  • $destroy() Destroy component/app

Vue对象选项

  • data() Init reactive data
  • props Data visible by parent
  • mixins Declares mixins
  • components Registers children
  • methods Set of Vue methods
  • watchers Watch values for change
  • computed Cached reactive methods

计算属性

computed: {
	fullname: function(){
        return this.fname + this.lname
    }
}

监听器

watch: {
    fname: function(newValue, oldValue){
        this.msg = 'fname changed'
    }
}

Tip

触发视图的更新

  1. 更新的类型

    直接赋值会更新。

    this.heros = []

    通过函数更新

    this.heros.push('xxx')

  2. 针对数组中使用函数进行更新

比如push,pop,shift,unshift,splice,reverse,sort

  1. 通过Vue.set来实现。
