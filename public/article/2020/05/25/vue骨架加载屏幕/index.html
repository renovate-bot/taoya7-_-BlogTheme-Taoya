创建一个显示用户信息的异步组件

ProfileCard.vue 创建一个组件显示作者默认信息。

因为这是一个异步组件。所以需要设置方法为异步方法。

<template>
    <div class="profile-card" v-if="user">
        <div class="profile_img">
            <img :src="user.avatar">
        </div>
        <div class="profile_info">
            <span>信息</span>
            <h3>{{user.name}}</h3>
            <p>{{user.sign}}</p>
        </div>
    </div>
</template>

<script>
    export default {
        name: "ProfileCard",
        data(){
            return {
                user:''
            }
        },
        methods:{
         loadUserData: async  ()=>{
            return new Promise(resolve=>{
                setTimeout(()=>{
                    resolve({
                        avatar: 'http://alicdn.itaolaity.com/img/20200419230410.jpg',
                        name: 'Taoya',
                        sign: '多言数穷，不如守中'
                    })
                }, 1500)
            })
         }
        },
        async mounted() {
            this.user = await this.loadUserData();
        }
    }
</script>

<div class="profile-card" v-else>
    <ContentLoader>

    </ContentLoader>
</div>

上面是最简单的解决方案。

但控制外观的最佳方式是为每一个元素设置样式。

有两种方法。

  1. 自己写布局样式。确保与实际组件具有相同的轮廓
    1. 灰色背景
    2. 动画
  2. 在线生成。然后将生成的放入到组件内。具体可查看文档。
